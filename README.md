## 소감
처음으로 intellij를 사용하여 개발하였는데 깔끔한 디자인과 기능 덕분에 새로운 기분으로 코딩할 수 있었다. 여러 개의 클래스와 인터페이스를 사용 관련한 자바 개념을 조금 더 보충시켜야 할 필요가 있다고 느꼈다.
자바를 복습하는 차원에서 프로젝트 1을 진행하였는데, 활용도가 높은 언어인 만큼 더 능숙하게 사용할 수 있도록 연습이 필요하다는 생각이 들었다. 교수님께서 설명해주시는 내용을 토대로 CRUD의 함수 형식을 어떻게 설계해야 할지 학습할 수 있었던 기회였던 것 같다.
